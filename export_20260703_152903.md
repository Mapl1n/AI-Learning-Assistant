# 专升本2026年高等数学大题考点精析

## 一、函数、极限与连续（★★★）
*   **核心大题方向**：极限的计算、间断点的判定与分类、闭区间上连续函数的性质（零点定理证明）。
*   **考点1：极限四则运算法则与重要极限（★★★）**
    *   0/0型、∞/∞型极限（洛必达法则前提）。
    *   1^∞型极限（重要极限公式：\(\lim_{x \to 0} (1+x)^{1/x} = e\)）。
    *   等价无穷小替换（\(x \to 0\) 时，\(\sin x \sim x, \tan x \sim x, \sqrt{1+x}-1 \sim \frac{x}{2}\) 等）。
*   **考点2：洛必达法则的适用条件与陷阱（★★★）**
    *   必须能化为0/0或∞/∞。
    *   非未定式不能直接洛必达。
*   **考点3：间断点的类型判断与计算（★★）**
    *   考察点：要能计算左右极限，区分第一类（可去/跳跃）与第二类（无穷/振荡）间断点。
*   **考点4：零点定理的应用（证明题）（★★）**
    *   构造辅助函数，利用区间端点函数值异号，证明方程有根。

## 二、一元函数微分学（★★★★★）
*   **核心大题方向**：导数计算、切线/法线方程、单调性/极值、最值应用题、中值定理证明。
*   **考点5：隐函数求导与对数求导法（★★★）**
    *   对隐函数方程两边求导，处理幂指函数（\(y = x^{\sin x}\)）。
*   **考点6：切线方程与法线方程（★★★）**
    *   求曲线在一点处的斜率，写出点斜式方程。
*   **考点7：利用导数研究函数单调性与极值（★★★）**
    *   求驻点、不可导点，列表判断单调区间与极值点（边界值求最值）。
*   **考点8：函数的凹凸性与拐点（★★）**
    *   求二阶导数为零或不存在点，判断凹凸区间与拐点。
*   **考点9：最值应用题（必考大题题型）（★★★★★）**
    *   **高频模型**：面积/体积最大最小（围墙、容器）、利润最大、用料最省。
    *   解题步骤：设未知数 -> 建立目标函数并化简 -> 求导 -> 找驻点 -> 判断最值。
*   **考点10：拉格朗日中值定理与柯西中值定理（证明题）（★★★）**
    *   常用于证明不等式（如 \(|\sin a - \sin b| \le |a-b|\)）或等式。
    *   难点：构造合适的辅助函数 \(F(x)\)，验证满足定理条件。

## 三、一元函数积分学（★★★★★）
*   **核心大题方向**：不定积分计算、定积分计算与面积、旋转体体积、反常积分收敛性。
*   **考点11：不定积分的基本解法（★★★）**
    *   凑微分法（第一类换元）：核心是配出 \(f(u)du\) 的形式。
    *   第二类换元法（三角代换、根式代换）：注意反函数回代。
    *   分部积分法（\( \int u dv = uv - \int v du \)）：常见口诀“反对幂三指”。
*   **考点12：定积分的计算（★★）**
    *   牛顿-莱布尼茨公式。
    *   利用对称性（奇函数 \([-a,a]\) 积分为0，偶函数翻倍）。
*   **考点13：定积分求平面图形面积（★★★★★）**
    *   X型与Y型积分区域划分。关键是确定被积函数是被减函数（上函数 - 下函数，右函数 - 左函数）。
*   **考点14：定积分求旋转体体积（★★★★）**
    *   **绕x轴旋转**：\(\pi \int_a^b [f(x)]^2 dx\)（圆盘法）。
    *   **绕y轴旋转**：\(\pi \int_c^d [\phi(y)]^2 dy\) 或 \(2\pi \int_a^b x f(x) dx\)（柱壳法/壳层法）。
    *   **较易出错点**：区分绕x轴与绕y轴的公式，注意是否需要切割成多个部分。
*   **考点15：反常积分（广义积分）审敛与计算（★★）**
    *   无穷限积分：上/下限为∞，计算极限。
    *   瑕积分：被积函数在某点无界（瑕点），需要分段。
    *   审敛：比较审敛法的应用（p-积分 \(\int_1^{\infty} \frac{1}{x^p} dx\)）。

## 四、常微分方程（★★★★★）
*   **核心大题方向**：一阶微分方程求解、二阶常系数线性微分方程求解。
*   **考点16：一阶微分方程的求解（★★★★）**
    *   **可分离变量型**：分离变量后两边积分。
    *   **齐次型**：令 \(u = y/x\)，化为可分离变量。
    *   **一阶线性非齐次**：套公式 \(y = e^{-\int P(x) dx} [\int Q(x) e^{\int P(x) dx} dx + C]\)。
*   **考点17：二阶常系数线性微分方程（★★★★★）**
    *   **齐次解**：由特征方程 \(r^2 + pr + q = 0\) 的根确定。
    *   **特解设定**（最重要的难点）：
        *   自由项 \(f(x) = P_m(x) e^{\lambda x}\) 时，特解设为 \(y^* = x^k Q_m(x) e^{\lambda x}\)，k为λ重合特征根的次数（0,1,2）。
        *   自由项 \(f(x) = e^{\alpha x} [A \cos \beta x + B \sin \beta x]\) 时，特解设为 \(y^* = x^k e^{\alpha x} [A_1 \cos \beta x + B_1 \sin \beta x]\)，k取决于 \(\alpha \pm i\beta\) 是否是特征根。
*   **考点18：微分方程的实际应用（综合题）（★★★）**
    *   结合几何应用（切线斜率）、物理模型（冷却定律、混合问题等）。

## 五、多元函数微分学（★★★）
*   **核心大题方向**：偏导数计算、全微分、条件极值（拉格朗日乘数法）。
*   **考点19：多元复合函数求导（链式法则）（★★★）**
    *   明确函数依赖关系图（树图），分清中间变量与自变量。
*   **考点20：隐函数求偏导（★★★）**
    *   利用公式法：\(\frac{\partial z}{\partial x} = -\frac{F_x}{F_z}\)。
*   **考点21：条件极值（乘数法）（★★★★★）**
    *   **经典题型**：某长方体的表面积固定，求体积最大值；或给定目标函数与约束条件。
    *   步骤：设辅助函数 \(F(x,y,\lambda) = f(x,y) + \lambda \phi(x,y) \)，解方程组找驻点，判断极值。

## 六、空间解析几何与向量代数（★★）
*   **核心大题方向**：平面方程、直线方程、距离计算。
*   **考点22：平面方程与直线方程（★）**
    *   点法式、一般式、截距式。
    *   直线的对称式、参数式。
*   **考点23：点到平面的距离、点到直线的距离（★★）**
    *   记住公式，代入计算即可。

## 七、无穷级数（★★★）
*   **核心大题方向**：级数审敛、幂级数展开（麦克劳林/泰勒）、求和函数。
*   **考点24：常数项级数审敛法（★★★）**
    *   正项级数：比值审敛（达朗贝尔）、根值审敛（柯西）、比较审敛（极限形式）。
    *   交错级数：莱布尼茨审敛（单调递减趋于0）。
    *   绝对收敛与条件收敛的区别。
*   **考点25：幂级数的收敛半径与收敛域（★★）**
    *   先求收敛半径（系数比值的极限），再单独分析端点。
*   **考点26：求幂级数的和函数（★★★）**
    *   常用技巧：逐项求导、逐项积分，结合已知展开式（如 \(\frac{1}{1-x}\)）。
    *   高阶考点：先求和函数，再求某些数项级数的和。

## 总结：2026大题必考题型（按频次排列）

1.  **【第1梯队】★★★★★（每年必出）**
    *   导数应用（最值应用题）
    *   定积分求面积/旋转体体积
    *   微分方程求解（特别是二阶常系数非齐次）
    *   条件极值（拉格朗日乘数法）
    *   **中值定理证明题**（抽中概率约60%）

2.  **【第2梯队】★★★★（高频，隔年出现）**
    *   幂级数求和函数或展开
    *   极限计算（含洛必达与等价无穷小）
    *   多元复合函数求偏导

3.  **【第3梯队】★★★（中频，易失分点）**
    *   不等式证明（构造函数）
    *   常数项级数审敛
    *   空间曲线/曲面切平面与法线
